# /break-even - 損益平衡分析命令

執行 Taiwanway 餐廳的損益平衡分析，評估不同成本情境下的盈虧狀況。

## 使用方式

```
/break-even [數據檔案路徑] [--labor 人力成本] [--food-rate 食材成本率]
```

## 參數說明

- `數據檔案路徑`：Square 支付數據 CSV 檔案（預設：`data/all_payments/all_payments.csv`）
- `--labor`：每日人力成本（預設：$160）
- `--food-rate`：食材成本率（預設：35%）

## 成本參數

### 固定成本（月）
| 項目 | 金額 |
|-----|------|
| 租金 | $3,100 |
| 水電 | $700 |
| **合計** | **$3,800** |

### 人力成本情境（日）
| 情境 | 金額 | 說明 |
|-----|------|------|
| 最低 | $100 | 店主獨自經營 |
| 標準 | $160 | 1-2 位員工 |
| 繁忙 | $200 | 週末或尖峰 |
| 尖峰 | $250 | 節日或特殊活動 |

### 食材成本率
| 情境 | 比率 | 說明 |
|-----|------|------|
| 優化 | 30% | 成本控制良好 |
| 標準 | 35% | 一般狀況 |
| 偏高 | 40% | 成本壓力大 |

## 執行流程

請使用 Break-Even Orchestrator 代理協調以下流程：

### Step 1: 數據準備 (break-even/data-prep)

1. 載入營收數據
2. 計算月度統計
3. 準備成本參數

### Step 2: 損益平衡計算 (break-even/calculation)

使用公式計算各情境損益平衡點：

```
月損益平衡 = (固定成本 + 月人力成本) / (1 - 食材成本率)
日損益平衡 = 月損益平衡 / 營業天數
```

### Step 3: 敏感度分析 (break-even/sensitivity)

建立 4×3 敏感度矩陣：
- 4 種人力成本情境
- 3 種食材成本率

計算：
- 各因素敏感度係數
- 安全邊際
- 風險等級

### Step 4: 利潤目標分析 (break-even/profit-target)

計算達成不同利潤目標所需營收：
- $1,000/月
- $2,000/月
- $3,000/月
- $5,000/月

### Step 5: 視覺化 (break-even/visualization)

生成以下圖表：
1. 損益平衡分析圖
2. 成本結構分解圖
3. 敏感度熱力圖
4. 情境比較柱狀圖
5. 月度損益比較圖

### Step 6: 報告生成 (break-even/report)

輸出：
- JSON 報告：`agents/output/break_even/break_even_report_{timestamp}.json`
- Markdown 報告：`agents/output/break_even/break_even_summary_{timestamp}.md`
- 圖表目錄：`agents/output/break_even/charts/`

## 輸出範例

```markdown
# 損益平衡分析報告

**分析日期**：2025-11-16
**數據範圍**：2025-09-01 ~ 2025-11-15

## 關鍵指標

| 指標 | 數值 | 狀態 |
|-----|------|------|
| 標準配置日損益平衡 | $612 | 基準 |
| 當前日均營收 | $753 | 盈利 |
| 安全邊際 | 18.7% | 中等風險 |

## 敏感度矩陣（日損益平衡）

|         | 30% 食材 | 35% 食材 | 40% 食材 |
|---------|----------|----------|----------|
| $100 人力 | $477 | $519 | $567 |
| $160 人力 | $563 | $612 | $669 |
| $200 人力 | $621 | $673 | $736 |
| $250 人力 | $693 | $750 | $819 |

## 風險評估

- **低風險情境**：15 種組合
- **高風險情境**：5 種組合
- **最敏感因素**：營業天數 > 食材成本率 > 人力成本

## 營運建議

1. **高優先級**：監控 11 月營收趨勢
2. **中優先級**：控制食材成本率在 35% 以下
3. **低優先級**：考慮淡季促銷活動
```

## 進階選項

### 自訂固定成本

```
/break-even --fixed-costs '{"rent": 3200, "utilities": 800}'
```

### 自訂營業天數

```
/break-even --operating-days 18
```

### 僅執行特定分析

```
/break-even --only sensitivity
/break-even --only profit-target
```

## 錯誤處理

- **營收數據不足**：至少需要一個月完整數據
- **參數超出範圍**：提供合理範圍建議
- **圖表生成失敗**：繼續文字報告，記錄錯誤

## 相關命令

- `/analyze`：完整營收分析
- `/report`：報告生成
