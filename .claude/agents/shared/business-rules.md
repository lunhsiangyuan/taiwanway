### 營業日
- **週一 (0)**：營業
- **週二 (1)**：營業
- **週三 (2)**：休息
- **週四 (3)**：休息
- **週五 (4)**：營業
- **週六 (5)**：營業
- **週日 (6)**：休息

程式碼表示：`operating_days: [0, 1, 4, 5]`

### 營業時段
- **開始時間**：10:00
- **結束時間**：20:00

### 休息月份
- **6 月**：暑期休息
- **7 月**：暑期休息

程式碼表示：`closed_months: [6, 7]`

### 特殊假日
- **12 月 25 日**：聖誕節休息

## 財務參數

### 稅率
- **NYC 銷售稅**：8.875%
  - 紐約州稅：4.0%
  - 紐約市稅：4.5%
  - MTA 附加稅：0.375%

### ⚠️ 營收計算（強制規則）

**重要**：所有營收相關分析**必須**使用**淨營收（Net Revenue）**，而非含稅金額（Gross Amount）。

```python
# 常數定義
NYC_TAX_RATE = 0.08875

# 淨營收計算公式
Net_Revenue = Gross_Amount / (1 + NYC_TAX_RATE)

# 範例：含稅 $100 的淨營收
net = 100 / 1.08875  # = $91.85
```

**適用範圍**：
- ✅ 月度/日度營收統計
- ✅ YoY 同期比較
- ✅ 營收預測
- ✅ 損益平衡分析
- ✅ 客戶消費金額分析
- ✅ 時段營收分析

**不適用**：
- ❌ 原始交易記錄（保留原始 amount）

### 數據欄位命名規範

| 欄位名稱 | 說明 | 計算方式 |
|----------|------|----------|
| `amount` | 原始含稅金額 | 保持原值 |
| `Net_Revenue` | 淨營收（稅後） | `amount / (1 + 0.08875)` |
| `Tax_Amount` | 稅金 | `amount - Net_Revenue` |

### 損益平衡參數

#### 固定成本（月）
| 項目 | 金額 |
|------|------|
| 房租 | $3,100 |
| 水電 | $700 |
| **合計** | **$3,800** |

#### 人力成本情境（日）
| 情境 | 金額 |
|------|------|
| 最低 | $100 |
| 標準 | $160 |
| 中等 | $200 |
| 最高 | $250 |

#### 食材成本率
| 情境 | 比率 |
|------|------|
| 低 | 30% |
| 中 | 35% |
| 高 | 40% |

#### 損益平衡公式
```
損益平衡營收 = (固定成本 + 人力成本) / (1 - 食材成本率)
```

## 時區設定

- **標準時區**：America/New_York
- **夏令時（EDT）**：UTC-4（3月第二週日 ~ 11月第一週日）
- **標準時間（EST）**：UTC-5（11月第一週日 ~ 3月第二週日）

**重要**：所有時間分析必須使用 pytz 進行自動 DST 處理。

## 數據過濾規則

### 交易狀態
只處理 `status == 'COMPLETED'` 的交易

### 去重規則
依 `payment_id` 或 `Transaction ID` 去重

### 營業日過濾
只包含營業日（週一、二、五、六）的數據

### 月份過濾
排除休息月份（6、7月）的數據
