╔══════════════════════════════════════════════════════════════════════════════╗
║                           關鍵演算法說明                                        ║
╚══════════════════════════════════════════════════════════════════════════════╝

┌──────────────────────────────────────────────────────────────────────────────┐
│ 演算法 1: VIP 客戶識別                                                         │
│                                                                              │
│ 輸入: DataFrame with Customer ID, Transaction ID, Net Sales                │
│                                                                              │
│ 步驟:                                                                        │
│ 1. customer_stats = df.groupby('Customer ID').agg({                        │
│        'Transaction ID': 'nunique',  # 造訪次數                             │
│        'Net Sales': 'sum'            # 總消費                               │
│    })                                                                        │
│                                                                              │
│ 2. customer_stats['AvgSpent'] = TotalSpent / VisitCount                    │
│                                                                              │
│ 3. vip_mask = (VisitCount > 10) | (TotalSpent > 200)                       │
│                                                                              │
│ 4. vip_customers = customer_stats[vip_mask].sort_values('TotalSpent')     │
│                                                                              │
│ 輸出: VIP 客戶 DataFrame                                                     │
│                                                                              │
│ 時間複雜度: O(n log n) - 由排序決定                                           │
│ 空間複雜度: O(k) - k = 唯一客戶數                                            │
└──────────────────────────────────────────────────────────────────────────────┘


┌──────────────────────────────────────────────────────────────────────────────┐
│ 演算法 2: 商品組合分析                                                         │
│                                                                              │
│ 輸入: DataFrame with Transaction ID, Item                                  │
│                                                                              │
│ 步驟:                                                                        │
│ 1. transaction_items = df.groupby('Transaction ID')['Item'].apply(list)   │
│                                                                              │
│ 2. multi_item_tx = transaction_items[transaction_items.apply(len) > 1]    │
│                                                                              │
│ 3. FOR each transaction IN multi_item_tx:                                  │
│        items = sorted(set(transaction))  # 去重並排序                       │
│        FOR each combination of 2 items:   # C(n, 2)                        │
│            combos.append(item1 + ' + ' + item2)                            │
│                                                                              │
│ 4. combo_counts = Counter(combos)                                          │
│                                                                              │
│ 5. RETURN combo_counts.most_common(top_n)                                  │
│                                                                              │
│ 範例:                                                                        │
│   Transaction 1: [麵, 飯, 飲料]                                              │
