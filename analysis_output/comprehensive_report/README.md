## 核心建議摘要

### 🎯 立即行動項目（0-30 天）

1. **建立會員系統**
   - 導入 Square Loyalty 或同類產品
   - 設計會員等級與權益
   - 目標：招募 100 位會員

2. **優化人力配置**
   - 尖峰時段增加人手
   - 離峰時段彈性排班
   - 預期節省人力成本 15-20%

3. **推出套餐組合**
   - 經典組合 $18-22
   - 提升客單價 15%

4. **啟動社群行銷**
   - 建立 Instagram/Facebook 官方帳號
   - 每日發佈內容
   - 舉辦打卡送飲料活動

### 📈 關鍵目標

| 指標 | 當前 | 3 個月目標 | 12 個月目標 |
|------|------|------------|-------------|
| 月營收 | $10,000 | $13,000 | $18,000 |
| 客單價 | $10-11 | $12-13 | $14-15 |
| 會員數 | - | 150 | 500 |
| 營業日 | 16 天 | 18 天 | 20 天 |
| 淨利率 | 虧損 | 打平 | 15-20% |

### 💡 突破性策略

1. **增加營業日**：從 16 天增至 20 天，分攤固定成本
2. **外送平台合作**：觸及 3-5 公里範圍新客源
3. **企業訂餐服務**：穩定離峰時段營收
4. **會員訂閱制**：推出「月享卡」創造穩定現金流
5. **零售商品**：茶葉、鳳梨酥等伴手禮增加營收來源
