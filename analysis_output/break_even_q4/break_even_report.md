## 四、損益平衡分析

### 損益平衡公式

```
損益平衡日營收 = (固定成本 + 月人力成本) / (1 - 食材成本率) / 月營業天數
```

## 五、結論與建議

### 關鍵發現

1. **營收表現**
   - Q4 日均營收 $799，表現穩定
   - 9月旺季 ($819) 明顯高於11月淡季 ($730)

2. **成本結構**
   - 固定成本佔比較高，需維持足夠營業天數攤薄
   - 人力成本對損益平衡點影響顯著

3. **損益平衡**
   - 以 35% 食材成本、$160/天人力計算，需約 25-27 天達成平衡
   - 每週營業 4 天（16 天/月）難以達成損益平衡

### 營運建議

1. **短期**
   - 維持每週 4-5 天營業
   - 控制食材成本在 35% 以下
   - 優化人力調度

2. **中期**
   - 提升日均營收至 $800+
   - 開發高毛利產品
   - 考慮增加營業時段

3. **長期**
   - 建立穩定客源降低季節波動
   - 評估營業天數與利潤最佳化組合

## 六、人力成本敏感度分析

### 人力成本對損益的影響

人力成本是影響損益平衡的關鍵變數之一。以下分析不同人力成本情境：

| 人力成本 | 說明 | 適用情境 |
|---------|------|---------|
| $100/天 | 最低配置 | 淡季或短時營業 |
| $160/天 | 標準配置 | 正常營業日 |
| $200/天 | 加強配置 | 週末或尖峰時段 |
| $250/天 | 完整配置 | 特殊活動或滿載 |

### 敏感度分析結論

- 人力成本每增加 $50/天，損益平衡天數增加約 3-4 天
- 在淡季，應考慮降低人力配置以維持獲利
- 旺季可適度增加人力，提升服務品質和翻桌率
