### 8. profit_target_days.png - 目標利潤所需營業天數（新增）

**4×3 子圖矩陣**，回答「每月要賺 $5,000 需要營業幾天？」

**計算公式**：
```
天數 = ($5,000 + $3,800) / [日均營收 × (1 - 食材率) - 人力/天]
     = $8,800 / 日淨貢獻
```

**矩陣結構**：
- **橫軸（3 列）**：淡季 $640、平均 $753、旺季 $820
- **縱軸（4 行）**：人力 $100、$160、$200、$250/天
- **每格內**：30%、35%、40% 三種食材成本率的柱狀圖

**顏色說明**：
- 🟢 綠色：≤20 天（輕鬆達成）
- 🟠 橙色：21-25 天（可行但緊繃）
- 🔴 紅色：26-31 天（勉強可行）
- ⬜ 灰色：>31 天（不可能）
- 紅色虛線：31 天上限

**關鍵結論（以 Q4 日均 $753、食材 35%）**：
- 人力 $100/天 → 需 22.6 天
- 人力 $160/天 → 需 26.7 天
- 人力 $200/天 → 需 30.4 天
- 人力 $250/天 → 不可能

## 關鍵發現

### 損益平衡天數摘要（日均 $753）

| 人力成本 | 淡季 $640 | 平均 $753 | 旺季 $820 |
|---------|----------|----------|----------|
| $100/天 | 10.9-13.5 天 🟢 | 9.0-10.9 天 🟢 | 8.2-9.8 天 🟢 |
| $160/天 | 13.5-17.0 天 🟢🟠 | 10.6-13.4 天 🟢 | 9.4-11.6 天 🟢 |
| $200/天 | 15.6-20.8 天 🟠🔴 | 11.8-15.6 天 🟢🟠 | 10.5-13.4 天 🟢 |
| $250/天 | 19.6-28.8 天 🟠🔴 | 13.9-19.6 天 🟢🟠 | 11.9-15.9 天 🟢🟠 |

### 經營建議

1. **淡季風險管理**
   - 人力 $250/天 + 40% 食材：需 28.8 天才能打平（不可能）
   - 建議淡季控制人力 ≤$160/天、食材 ≤35%

2. **安全邊際**
   - 目前每月營業約 17 天
   - 人力 $160/天 + 35% 食材：只需 11.7 天即可打平
   - 安全邊際：5.3 天（約 31%）

3. **旺季優勢**
   - 日均 $820 時，所有合理成本組合都在 16 天內可打平
   - 可承受更高的人力成本投入
