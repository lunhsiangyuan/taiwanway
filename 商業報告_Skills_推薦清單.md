# 商業報告產出 Skills 推薦清單

## 📊 核心文件處理 Skills（必備）

### 1. **pptx** - PowerPoint 簡報生成
- **GitHub**: `anthropics/skills/tree/main/document-skills/pptx`
- **功能**：
  - 建立專業 PowerPoint 簡報
  - 支援多種版面配置和範本
  - 自動生成圖表、時間軸、定價表
  - 品牌化設計（符合公司風格）
- **適用場景**：
  - 商業簡報製作
  - 季度報告展示
  - 客戶提案
  - 董事會報告

### 2. **xlsx** - Excel 試算表處理
- **GitHub**: `anthropics/skills/tree/main/document-skills/xlsx`
- **功能**：
  - 建立和編輯 Excel 檔案
  - 公式計算和數據分析
  - 圖表視覺化
  - 數據轉換和清理
- **適用場景**：
  - 財務報表
  - 數據分析報告
  - KPI 儀表板
  - 預算規劃

### 3. **docx** - Word 文件處理
- **GitHub**: `anthropics/skills/tree/main/document-skills/docx`
- **功能**：
  - 建立專業 Word 文件
  - 追蹤變更和註解
  - 格式化保留
  - 文字提取和分析
- **適用場景**：
  - 詳細商業報告
  - 提案文件
  - 執行摘要
  - 正式文件

### 4. **pdf** - PDF 處理
- **GitHub**: `anthropics/skills/tree/main/document-skills/pdf`
- **功能**：
  - 提取文字和表格
  - 合併/分割 PDF
  - 建立新 PDF
  - 處理表單
- **適用場景**：
  - 報告匯出
  - 文件合併
  - 資料提取

## 📚 推薦組合方案

```
5. brand-guidelines (品牌化)
```

## 🚀 安裝方式

### Claude.ai Web 介面
1. 前往 [Settings > Capabilities](https://claude.ai/settings/capabilities)
2. 啟用 Skills 功能
3. 瀏覽可用 Skills 或上傳自訂 Skills

### Claude Code CLI
```bash
# 從 marketplace 安裝
/plugin marketplace add anthropics/skills

# 或從本地目錄安裝
/plugin add /path/to/skill-directory
```

### 手動安裝（Claude Code）
```bash
# 建立 Skills 目錄
mkdir -p ~/.config/claude-code/skills/

# 複製 Skill 到目錄
cp -r skill-name ~/.config/claude-code/skills/

# 驗證 Skill 元數據
head ~/.config/claude-code/skills/skill-name/SKILL.md
```

## 💡 使用建議

1. **開始簡單**：先從 `pptx` 和 `xlsx` 開始，這是最常用的
2. **品牌一致性**：使用 `brand-guidelines` 確保所有報告符合公司風格
3. **視覺化優先**：結合 `scientific-visualization` 和 `D3.js` 建立專業圖表
4. **多格式輸出**：同時生成 Word、PowerPoint 和 PDF 版本
5. **互動式展示**：使用 `artifacts-builder` 建立網頁版報告

## ⚠️ 注意事項

- Skills 可以執行任意代碼，請只從信任的來源安裝
- 安裝前請仔細檢查 SKILL.md 和所有腳本
- 建議在非生產環境先測試
- 定期更新 Skills 以獲得最新功能
