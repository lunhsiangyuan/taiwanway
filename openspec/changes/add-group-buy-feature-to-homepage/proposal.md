## Why

為提升 `taiwanway` 網站的用戶參與度與營收潛力，我們需要將現有的團購功能在首頁上更加突出地展示。目前團購資訊可能不夠顯眼，導致潛在用戶難以發現並參與，進而影響轉換率。此變更旨在改善用戶體驗，並透過增加團購功能的曝光度來促進銷售。

## What Changes

*   **新增**：在網站首頁上新增一個專屬區塊，用於展示活躍中的團購活動。
*   **修改**：現有的團購功能將保持不變，但其入口與展示方式會與首頁新區塊整合。
*   **移除**：無。

## Capabilities

### New Capabilities
*   `homepage-group-buy-display`: 在首頁上展示當前活躍團購活動的列表或卡片。
*   `group-buy-google-sync-integration`: 確保團購活動資訊能與 Google Sheets 或相關服務同步，以便在首頁展示最新資料。

### Modified Capabilities
*   `group-buy-listing`: 團購列表的資料來源和篩選邏輯可能需要優化，以支援首頁展示的性能和資料即時性。

## Impact

*   **前端**：
    *   `src/components/Homepage.js` 或相關首頁模板文件將會修改，以整合新的團購展示區塊。
    *   可能需要新增 `GroupBuyCard` 或 `GroupBuyList` 等新組件。
    *   CSS/SCSS 樣式文件將會更新，以確保首頁團購區塊的視覺呈現。
*   **後端/API (若有)**：
    *   若目前團購資料非直接由前端處理，可能需要新增或修改 API 端點，以提供首頁展示所需的團購數據（例如，只返回活躍團購、精簡數據）。
*   **資料同步**：
    *   需要驗證或建立機制，確保 Google Sheets 中的團購數據能有效且即時地更新至網站前端展示。
*   **性能**：首頁加載時間可能因新增內容而略有增加，需要進行性能優化。
*   **測試**：需要為新的首頁團購區塊和資料整合進行單元測試、整合測試和端對端測試。
